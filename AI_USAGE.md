# AI Usage

## Overview

Artificial Intelligence tools were used throughout the development process to improve productivity, accelerate development, and assist with debugging and validation were performed.

## AI Tools Used

### ChatGPT

ChatGPT was used to assist with:

* Planning the overall project architecture.
* Designing the frontend and backend folder structure.
* Reviewing implementation approaches.
* Suggesting UI layouts and component organization.

### OpenCode

OpenCode was used to assist with:

* Debugging application errors during development.
* Identifying and resolving frontend and backend issues.
* Reviewing code for potential improvements.
* Verifying API integration and application flow.
* Assisting with project structure and development workflow.

## AI-Assisted Development Areas

The following areas benefited from AI assistance:

* Project planning and architecture
* Frontend component structure
* Backend API organization
* Authentication workflow
* Document upload and processing
* Open Router API integration
* Debugging and error resolution
* Documentation and project organization

## Developer Responsibility

AI tools were used as development assistants. All code was reviewed, modified where necessary, tested locally, and integrated into the application by the developer. Final implementation decisions and validation were performed manually to ensure the application met the project requirements.

## Summary

AI assistance helped improve development efficiency by reducing debugging time, organizing the project structure, and providing guidance during implementation while maintaining full developer oversight throughout the project.
